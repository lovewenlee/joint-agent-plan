# 团队建设与制度保障

## 组织架构设计

### 设计原则
- **小而美**：避免大公司病，保持创业团队活力
- **数据驱动**：基于数据做决策，量化考核

## 核心岗位招聘标准

### 算法总监
**要求**：
- 博士学历，计算机/AI/数学相关专业

**薪酬包**：80-100万/年（基础薪资70% + 项目奖金30%）

**要求**：
- 硕士学历，3年以上B端产品经验
- 熟悉SaaS产品设计和运营
- 有能源、IoT或工业软件经验
- 优秀的客户沟通和需求分析能力

**薪酬包**：60-80万/年（基础薪资70% + 项目奖金30%）

**要求**：
- 5年以上能源行业经验
- 熟悉光伏、储能、充电业务流程
- 优秀的方案设计和客户交流能力
- 有大项目实施和交付经验

**薪酬包**：50-70万/年（基础薪资60% + 项目奖金40%）

## 激励机制设计

### 薪酬结构
```
总薪酬包构成
├── 基础薪资（60-70%）：市场80分位水平
├── 项目奖金（20-30%）：基于项目成功交付
├── 年终奖金（5-10%）：基于年度KPI达成
└── 股权激励（长期）：虚拟股权，3年解锁
```

### 项目奖金机制
- **客户签约奖**：新客户签约，团队分享合同金额的2-5%
- **交付质量奖**：客户满意度90%+，项目团队奖励5-10万
- **创新突破奖**：技术突破或专利申请，个人奖励5-10万

### 股权激励计划
- **覆盖范围**：所有正式员工
- **激励池**：预留部门利润的20%作为虚拟股权池
- **解锁机制**：3年线性解锁，离职前解锁部分可兑现

### 非物质激励
- **技术成长**：年薪10%培训预算，支持参加顶会和培训
- **职业发展**：明确晋升通道，专家线和管理线双轨发展
- **工作环境**：灵活办公，高配硬件，舒适办公环境

## 制度保障体系

### 考核机制
```
分层考核体系
├── 部门级KPI（季度）
│   ├── 业务指标：客户数、收入、产品质量
│   ├── 技术指标：算法精度、系统稳定性
│   └── 管理指标：团队满意度、人才保留率
├── 团队级OKR（月度）  
│   ├── 算法团队：模型效果、开发效率
│   ├── 产品团队：交付质量、用户体验
│   └── 方案团队：客户成功、销售支持
└── 个人级绩效（双月）
    ├── 工作交付：具体任务完成情况
    ├── 协作能力：跨团队配合表现
    └── 创新贡献：技术突破和业务创新
```

### 资源保障
```
专项资源配置
├── 预算独立性：独立预算科目，直接向CEO汇报
├── 人事权限：100万以下招聘权限，特殊薪酬包授权
├── 采购权限：50万以下设备、软件采购授权
├── 对外合作：高校合作、专家顾问、外部服务
└── 技术资源：GPU计算资源、云服务、开发工具
```

### 决策流程
```
快速决策机制
├── 日常运营：部门负责人全权决定
├── 技术决策：技术委员会48小时快速评审
├── 人事决策：关键岗位72小时完成面试
└── 商务决策：CEO参与关键客户谈判
```

## 文化建设

### 核心价值观
- **客户成功**：客户的成功就是我们的成功
- **技术驱动**：用技术解决真实业务问题
- **开放协作**：开放透明，高效协作
- **持续学习**：保持好奇心，持续学习成长

### 团队文化活动
- **技术分享**：每周技术分享，跨团队学习交流
- **客户走访**：定期客户现场调研，了解真实需求
- **团建活动**：季度团建，增强团队凝聚力
- **创新大赛**：年度创新项目竞赛，鼓励技术创新

## 外部合作网络

### 技术合作
- **高校合作**：清华大学能源互联网研究院
- **研究机构**：中科院自动化所、电科院
- **技术顾问**：聘请3-5名行业技术专家

### 生态合作
- **开源社区**：参与PyTorch、TensorFlow等开源项目
- **行业组织**：加入中国储能联盟、充电联盟等
- **合作伙伴**：与系统集成商、设备厂商建立合作

### 人才培养
- **校企合作**：与顶级高校建立实习生计划
- **内推计划**：员工推荐奖励机制
- **雇主品牌**：参加技术会议，提升雇主品牌知名度
